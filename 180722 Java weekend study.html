2.     Try&Except

Try: 실행 코드를 실행하고 에러가 없을 시에 진행한다.

Except: 실행 코드에서 에러가 발생 시 Except로 진행한다.

Finally: 예외처리에서 무조건 실행되는 코드

Else: 에러 발생하지 않았을 시에 직접적 명령으로 인해 실행되는 명령
